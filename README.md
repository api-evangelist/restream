# Restream (restream)
Restream is a multistreaming service that allows content creators to  simultaneously broadcast live video to multiple platforms like YouTube,  Facebook, Twitch, LinkedIn, and more. The platform provides APIs for  managing streams, channels, and user accounts.


**URL:** [Visit APIs.json URL](https://restream.io)
